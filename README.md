为什么总是失败
- 信息 错误或不足
    - 错误
        - 有人传递了错误的信息给你
    - 不足
        - 有人屏蔽了一部分信息 或者说 有人只传递了一部分信息给你 ，而其它人却有完整的信息
        - 有人通过不公平的方式获得了更多的信息
        - 信息已经过时了，自己才获得
- 知识 错误或不足
    - 错误
        - 有人传递了错误的知识给你
    - 不足
        - 自己没有学会
        - 前人的总结不够全面
- 执行力 错误或不足
    - 自己的执行力不足
        - 生病了
        - 年龄大了
    - 执行的过程中出现了细节上的错误（就是自己不够细心）
    - 因为错误的知识或错误的信息导致了错误的执行
    - 有人在执行的过程中制造障碍（所以要闷声发大财）
- 成果被掠夺了（所以要尽力润到墙外）
- 自己的期望太高了（其实这个可以算在 知识的错误）
- 缺少必要的资源
- 其它无法分类的意外？
    - 工具出现故障
    - 流程出现无法恢复的中断（例如，因为疫情被隔离从而无法继续工作）

如何避免失败
- 信息使用多个信源，信息要交叉验证
- 知识要成体系，知识的学习要既有广度也有深度
- 不要被留意
- 尽量地低调
- 警惕消费主义
- 警惕各种投资骗局
- 从自己的实际出发不要攀比
- 从自己的实际出发建立期望
- 保持健康的生活方式，尽力预防生病
- 减少娱乐和无效社交

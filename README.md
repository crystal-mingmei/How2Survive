## 如何保证安全

- 不要有危险的行为
- 不要参与高危运动
- 不要去危险的地方旅游

## 如何保持身体健康

- 尽力保持健康的生活方式
    - 不要有良习惯，包括但不限于，吸烟，饮酒，槟榔
- 尽力避免生病
- 定时体检
    - 体检的项目？
- 生病了要及时就医
